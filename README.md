# ImuniKids 🧒💉

Aplicação desenvolvida como solução para o desafio técnico proposto pela Cyrrus, com o objetivo de auxiliar pais e responsáveis no acompanhamento da jornada de vacinação infantil através de uma experiência digital simples, intuitiva e responsiva.

## 🎯 Objetivo

O ImuniKids foi projetado para reduzir a dependência da carteira física de vacinação, permitindo o acompanhamento centralizado das informações vacinais das crianças de uma família.

A solução busca facilitar a identificação de vacinas aplicadas, vacinas pendentes, campanhas ativas e o acompanhamento individual de cada criança.

---

## 🚀 Funcionalidades Implementadas

### 👶 Gestão de Crianças

* Cadastro e acompanhamento de múltiplas crianças.
* Visualização individual do perfil de cada filho.
* Separação clara dos históricos vacinais.

### 💉 Controle Vacinal

* Listagem das vacinas previstas.
* Identificação visual de vacinas aplicadas.
* Identificação de vacinas pendentes.
* Destaque para vacinas em atraso.

### 📋 Histórico de Vacinação

* Consulta do histórico completo de vacinação.
* Registro das vacinas recebidas.
* Organização cronológica das informações.

### 📢 Campanhas de Vacinação

* Exibição de campanhas ativas.
* Destaque para campanhas relevantes ao público infantil.

### 📊 Situação Vacinal

* Indicadores visuais para facilitar o entendimento da situação vacinal.
* Resumo rápido do status de cada criança.

---

## ✅ Cenários Atendidos

### Cenário 1

O responsável consegue identificar facilmente quais vacinas já foram aplicadas e quais ainda precisam ser realizadas.

### Cenário 2

Vacinas com data prevista ultrapassada são destacadas visualmente para indicar pendências.

### Cenário 3

Campanhas de vacinação ativas são exibidas dentro da aplicação de forma clara e acessível.

### Cenário 4

Famílias com múltiplos filhos conseguem acompanhar cada criança individualmente, evitando confusão entre históricos e situações vacinais.

## ▶️ Executando Localmente

Clone o repositório:

```bash
git clone https://github.com/Viuutu/imunikids.git
```

Acesse a pasta:

```bash
cd imunikids
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
ionic serve
```
